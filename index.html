    GeoInfo

    Qu'est-ce que GeoInfo ?

    GeoInfo est un outil cartographique interactif optimisé pour la Suisse, et plus particulièrement pour le Canton de Vaud. Il permet d'identifier instantanément les appartenances administratives et territoriales d'un point géographique précis.

    Comment l'utiliser ?

    • Navigation libre : Déplacez la carte jusqu'au point souhaité. Les informations se mettent à jour automatiquement après une seconde d'arrêt.
    • Recherche : Utilisez la barre de recherche en haut pour trouver une adresse spécifique.
    • Ma position : Cliquez sur le bouton my_location pour centrer la carte sur votre position GPS actuelle.
    • Découvrir les limites : Pour la Commune, le District et le Canton, cliquez sur l'icône map dans les fiches d'info pour afficher les frontières administratives.
    • Copier les infos : Cliquez sur n'importe quelle fiche d'information pour copier son contenu dans votre presse-papier.

    Fonctionnalités avancées

    Basculez entre la vue plan et satellite via le bouton satellite_alt. Gérez l'affichage des informations grâce au menu des couches layers.

    Affichage des coordonnées en format WGS84 (GPS standard) et LV95 (coordonnées suisses).

    Projet personnel & Données publiques
    Les informations affichées proviennent de sources de données publiques ouvertes (OpenStreetMap, Nominatim, ESRI, Swisstopo). Ce projet est développé à titre personnel pour faciliter l'accès à l'information géographique.
